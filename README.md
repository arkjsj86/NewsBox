# NewsBox

## 프로젝트 목적

NewsBox는 특정 검색어와 관련된 최신 뉴스를 한곳에서 빠르게 확인할 수 있도록 돕는 웹 프로젝트다.

사용자가 예를 들어 `AI` 같은 검색어를 입력하면, 해당 주제와 관련된 뉴스를 수집하고 중복을 최대한 제거한 뒤 읽기 쉬운 리스트 형태로 보여주는 것을 목표로 한다.

이 프로젝트는 단순히 뉴스를 많이 보여주는 것이 아니라, 같은 내용의 기사가 여러 매체에 반복 노출되는 문제를 줄이고 핵심 기사 목록을 더 깔끔하게 제공하는 데 초점을 둔다.

## 핵심 목표

- 특정 키워드 기반으로 관련 뉴스 목록을 제공한다.
- 같은 기사 또는 거의 같은 기사를 최대한 중복 없이 정리한다.
- 최신 뉴스가 유지되도록 정기적으로 데이터를 갱신한다.
- 웹 사이트처럼 가볍고 빠르게 접근할 수 있는 형태로 제공한다.
- GitHub 리포지토리 기반으로 관리하고 배포 가능한 구조를 만든다.

## 기대 사용자 경험

1. 사용자가 키워드를 입력한다.
2. 시스템이 해당 키워드와 관련된 뉴스 데이터를 준비한다.
3. 중복도가 높은 항목을 제거하거나 묶는다.
4. 사용자는 최신순으로 정리된 뉴스 리스트를 본다.
5. 사이트의 뉴스 데이터는 약 1시간 단위로 갱신된다.

## 프로젝트 범위

현재 기준에서 NewsBox의 1차 목적은 아래에 집중한다.

- 키워드 기반 뉴스 탐색
- 뉴스 리스트 표시
- 중복 제거 로직
- 주기적 갱신
- GitHub 중심의 배포와 운영

## 아직 결정하지 않은 항목

아래 항목은 이후 설계 단계에서 확정한다.

- 뉴스 데이터를 가져오는 구체적인 소스
- 중복 제거 기준과 알고리즘
- 프론트엔드 프레임워크 사용 여부
- 저장 방식(JSON 파일, 캐시, 별도 DB 등)
- 검색 가능한 키워드 범위

## 제외 범위

초기 버전에서 아래 내용은 우선순위를 낮게 둔다.

- 뉴스 원문 전체 저장
- 회원가입 및 로그인
- 개인화 추천
- 복잡한 관리자 화면
- 대규모 실시간 검색 인프라

## 운영 방향

- 프로젝트는 GitHub 리포지토리에서 관리한다.
- 사이트는 웹에서 바로 접근 가능한 형태로 제공한다.
- 뉴스 데이터는 주기적으로 갱신되며, 초기 기준은 1시간 1회 갱신이다.
- 구현 방식은 나중에 바뀔 수 있어도, "키워드 기반 뉴스 수집 + 중복 제거 + 정기 갱신"이라는 핵심 목적은 유지한다.

## 한 줄 정의

NewsBox는 특정 키워드에 대한 최신 뉴스를 모아 중복을 줄이고, 주기적으로 갱신해 보여주는 뉴스 큐레이션 웹 프로젝트다.
